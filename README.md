## Software Architect

Software Architect at VMware, I leverage my 23+ years of experience in web application development, internet security management, and UI engineering to design and deliver innovative solutions for cloud computing, network function virtualization, and in-flight entertainment. I have a strong background in product engineering, project management, and team coordination, and I am always eager to take on new challenges and learn new technologies. In my previous roles, I have developed web-based test automation products and IFE products for avionics industry, worked with clients from various countries and domains, and contributed to multiple publications on web programming and JavaScript. My goal is to use my skills and knowledge to create secure, scalable, and user-friendly web applications that enhance the performance and productivity.

- 🌍  I'm based in Chennai, India
